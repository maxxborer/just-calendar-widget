# Support

- For reproducible bugs, use the [bug report form](https://github.com/maxxborer/just-calendar-widget/issues/new/choose).
- For feature ideas, use the [feature request form](https://github.com/maxxborer/just-calendar-widget/issues/new/choose).
- For security concerns, follow the [security policy](SECURITY.md).

Please include the macOS version, Just Calendar Widget version, and a clear description of what you expected to happen.
